Code Structure:

  • HTML:'Hypertext Markup Language' - Most current version is HTML5 - Used to create web pages
  • Block - Section of code which consists of one or more statements. Blocks can be nested within other blocks
  • Consistent indentation helps make the code more readable
  • div tags divide layout into major sections
  • Separate content (HTML) from Presentation (CSS)
  • "h" headers ascending in number descend in sub-headings
  • Use divs sparingly
  • CLOSE TAGS!
  • Use lowercase markup
  • Use alt attributes with images

Avoiding Repetition:

"Don't repeat yourself (DRY) also known as DIE Duplication is Evil'."

There is a lot of information about why it's important to keep things modular in programming. The shorter and simpler code is written, the more easily it is changed and modified for other pages. This also helps keeping the code clean for adding functionality until you need it.

CSS: 'Cascading Style Sheets'

  • Is code written to control the "style" of HTML elements.
  • It's a style sheet language
  • Different styles are supported to create style throughout the entire HTML or specific sections
  • It is to create separating between the content (in the HTML) and the style
  • To force background color of an h1 header you would type:
    • H1 {background-color: blue}
  • There are many different styles available from outline, images, text, etc. These are all in addition to format changes like alignment, columns, rows, etc.
